Estrutura Básica de HTML[Voltar]

Todo projeto em HTML tem como nome da página inicial, por concordancia virtual, o termo index.html.

A forma básica das paginas se dá em quatro tags principais.São elas:

  • Html: Contém as configurações da pagina
  • Head: O conteúdo aqui escrito não aparece na página. Aqui pode ser escrito também o nome a ser dado para a guia.
  • Body: Parte em que se encontra toda a estrutura visual do site. Aqui se encontram titulos, textos, fotos e etc.
  • Footer: Como o nome sugere, é o rodapé da pagina. Nele se encontram informações legais, pertinentes a todo o conteúdo da página, links para páginas importantes e outros recursos relevantes.

Tags e seu funcionamento[Voltar]

A definição mais simples de tags é que são elementos que marcam e formatam o conteúdo do HTML. Cada tag tem um nome e uma sintaxe a ser usada. Para determinar tem que ser aberta e fechada.

Exemplos de tags:

  • < p > Conteúdo < /p > : define um parágrafo;
  • < i > Conteúdo < /i > : deixa o conteúdo em itálico;
  • < strong > Conteúdo < /strong > : aplica efeito de negrito.

Atributos das Tags[Voltar]

As tags em HTML podem receber caracteristicas que modificam seu comportamento e que podem atribuir regras e delimitação.

Existem tags em que para inserir atributos a elas, é necessário coloca-los entre os '< >' de inicio.

Assim também existem atributos genéricos, que são utilizados em quase todas as tags(como id e class)e até especificos(como type).

Exemplo de Tags com Atributos:

  • < img src="link" > o atributo src define o link da imagem;
  • < input type="text" > o atributo type define o tipo do campo.

Textos[Voltar]

Fazendo parte da nomenclatura da tecnologia, aos textos também podem ser atribuidos estilizações para uma experiencia visual mais agradável.

Esses efeitos são:

  • Rabisco com < del >< /del >: Palavra rabiscada

  • Itálico com < i >< /i >: Palavra em itálico

  • Marcação com < mark >< /mark >: Palavra marcada

Listas Ordenadas e Não Ordenadas[Voltar]

Listas são basicamente o que chamamos de tabulação.

É inicialmente formada por uma tag "mãe", que define seu tipo, seguida de seus "filhos"

Listas Ordenadas:

Listas ordenadas são construídas a partir da tag < ol >< /ol >, e dentro dessa tag, os itens com a tag < li >< /li .

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenadas:

Listas não ordenadas são construídas a partir da tag < ul >< /ul >, e dentro dessa tag, os itens com a tag < li >< /li >.
  • Item 1
  • Item 2
  • Item 3

Links e suas características[Voltar]

O uso de link se dá pela necessidade de conectar paginas e/ou outros sites. A tag utilizada é < a >< /a ><. É necessário também atribuir "href" à tag e incluir o link de destino.

Exemplo:

< a href = "link"Clique aqui< /a >

Como abrir links em outras abas:

Clicando em um link, de forma padronizada, ele irá redirecionar a página atual. De forma a ficar mais organizado, existe uma forma de abrir em uma nova aba do navegador.Para isso, existe o atributo target que pode receber o valor _blank.

Exemplo:

< a href="link" target="_blank"Clique aqui< /a >

Ancoragem

Outra possibilidade criada por essa ferramenta é a Ancoragem. Através dela podemos conectar diferentes regiões da mesma página. Uma espécie de links de navegação.

Para "alocar" as sessões é necessário utilizar do simbolo # seguido do id da tag alvo. A mesma precisa ter o id, pois é necessário para a ancoragem
